Auto

Auto action directives are used to automatically change the state of the action.

Enabling/Disabling Behavior

Behavior can be disabled by passing in false to the directive.

fastTrigger

The fastTrigger and instantTrigger inputs exist to make it easy to set the trigger debounce and throttle times without explicitly passing them values.

The fast trigger is useful for autosave cases of forms. It has a 200ms debounce and 500ms throttle.

instantTrigger

The instant trigger is useful for cases where the action should go off immediately. It has no throttle and a minor debounce. An example is when an action recieve it's value from a user popup.
