Button

Mode set to button. The button will get it's icon from the current anchor, or default to the input anchor. Default text is used for states that do not have an anchor.
Rotate Button Mode

The above uses the "menu" button mode, which is the default.

There is an additional "rotate" button mode which shows the next anchor and when clicked will segue to that anchor instead.

Icon Button

When mode is set to icon only the icon is displayed. The icon input configures the navbar to only show that icon.
