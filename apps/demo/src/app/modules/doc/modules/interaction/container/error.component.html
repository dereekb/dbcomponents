[iconOnly]

dbx-error can also be simplified to showing an icon-only mode. This is useful for cases where screen space is limited. All errors, including those with custom inline error components, are reduced to only the icon.

Custom Widgets

Using the widget extension and dbxErrorWidgetService you can register custom errors widgets to display for given errors.

The presentation of an error shown by dbx-error can also be customized by registering an error code with an errorComponentClass provided.
