dbx-loading is typically used in conjunction with a LoadingContext, which streams LoadingContextEvents signifying the current loading state.

There are two presentations:

Default

Linear

color=accent

color=warn

Padding

The loading bar can be configured to provide padding to prevent content from jumping around when loading begins/ends. Useful in cases where the loading bar doesn't control any content but removing it would push the content upward to fill the empty space.

Custom content not pushed by the bar.
